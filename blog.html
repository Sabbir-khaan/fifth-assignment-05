Question-1: What are the different ways to select an element in the DOM?

There are 5 different ways to select an element in DOM

  1. 1. getElementsByClassName Selector
  2. 2. getElementsByTagName Selector
  3. 3. getElementById Selector
  4. 4. querySelector
  5. 5. querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  1. 1. innerHTML sets and returns the content inside an element
  2. 2. innerText sets and returns the text inside an element
  3. 3. textContent sets and returns all the text inside an element

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a way to handle events efficiently by adding a single event listener to a parent element instead of multiple child elements. When a child is clicked, the event "bubbles up" to the parent, allowing it to handle the action. This makes code simpler and improves performance.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM means when you click on an element, the event first happens on that element and then moves up to its parent elements. For example, if you click a button inside a div, the button event happens first, then the div event, and so on.

Question-5: How do you create, add, and remove elements using JavaScript?

You can create, Add, and remove element using JavaScript in 3 simple way.

  1. 1. Using the .createElement Selector to Create element
  2. 2. Using the .appendChild Selector to add element
  3. 3. Using the .removeChild Selector to remove element
